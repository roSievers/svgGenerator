Über das Projekt

Der SVG Generator ist ein Projekt von Rolf Sievers mit dem Ziel, das Erstellen von standardisierten Aufgabentypen zu erleichtern. Insbesondere soll dies möglich werden, ohne ein Grafikprogramm verwenden zu müssen.

Die primäre Zielgruppe ist das Grundschulaufgabenteam des Vereins MoNi, welcher sich um die Durchführung der Mathematik-Olympiade in Niedersachsen kümmert.

Dieses Projekt wird nicht von MoNi getragen, sondern von mir privat entwickelt. Für Fehlermeldungen und Anregungen erreichen Sie mich am besten über meinen Github Account roSievers oder per Mail an rolf.sievers beim deutschen Anbieter Posteo.

Das Projekt ist möglich durch freie Bibliotheken, hierbei handelt es sich um Bootstrap, eine sha1 Implementation von Chris Venes, FileSaver und Raphaël, welche alle unter der MIT- oder einer ähnlichen Lizenz verfügbar sind. Der Code dieses Projekts steht ebenfalls unter der MIT Lizenz.

Bei den verwendeten Icons handelt es sich um Glyphicon Halflings. Diese sind nicht frei, können aber zusammen mit Bootstrap kostenfrei verwendet werden.
